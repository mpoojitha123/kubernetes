how to create service account:

1. create cluster in kubernetes
2. connect to open lens via google cloud sdk shell terminal
3. create service account via open lens terminal
   kubectl -n kube-system create serviceaccount kubeconfig-sa
4. create cluster role binding
   kubectl create clusterrolebinding <binding-name> --clusterrole=cluster-admin --serviceaccount=kube-system:<service-account-name>
   eg: kubectl create clusterrolebinding add-on-cluster-admin --clusterrole=cluster-admin --serviceaccount=kube-system:kubeconfig-sa
5. yaml file need token from service account for that create a .yaml file.
   apiVersion: v1
   kind: Secret
   metadata:
    name: sa-token
    namespace: kube-system
    annotations:
      kubernetes.io/service-account.name: kubeconfig-sa
   type: kubernetes.io/service-account-token
